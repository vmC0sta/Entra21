# Sistema Simples de Login e Registro

Este é um sistema simples de login e registro implementado usando HTML, Bootstrap e JavaScript. Os usuários podem registrar seu email e senha e, em seguida, fazer login com as credenciais registradas. Além disso, há uma opção para remover os dados do usuário armazenados no armazenamento local.

## Funcionalidades

- Registro de usuário com email e senha.
- Funcionalidade de login do usuário.
- Armazenamento de dados no armazenamento local.
- Capacidade de remover dados do usuário.

## Iniciando

Para executar este projeto, siga estas etapas:

1. Clone este repositório: `git clone https://github.com/vmC0sta/sistema-login-registro.git`
2. Abra o arquivo `index.html` em um navegador da web.

Você também pode implantar este projeto em um servidor web para torná-lo acessível online.

## Uso

1. Abra a página da web em um navegador.
2. Insira seu email e senha para se registrar.
3. Clique no botão "Cadastrar" para registrar.
4. Para fazer login, insira seu email e senha registrados.
5. Clique no botão "Logar" para fazer login.
6. Você pode remover os dados do usuário armazenados clicando no botão "Remover".

## Tecnologias Utilizadas

- HTML
- Bootstrap
- JavaScript
